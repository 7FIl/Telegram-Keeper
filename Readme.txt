How to use:
1. Check for Depdency
2. Change api_id and api_hash inside api.json  into your api id and api hash that you can get from registering at https://my.telegram.org/apps
3. Fill the number.txt with your account number (Account that you want to auotomate)
4. Run the Script

Thats it, Enjoy

Depedency:

- telethon
- tqdm

If you dont have any of it you can install it by typing "pip install (depedency)" in your terminal
